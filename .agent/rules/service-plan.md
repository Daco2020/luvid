# Luvid: 연애 초보를 위한 따뜻한 가이드 플랫폼

Luvid는 **연애가 처음이거나 어려운 2030 청년**들이 자신을 이해하고, 관계를 시작하고, 불안을 극복할 수 있도록 돕는 따뜻한 동반자입니다.

## 타깃 유저

- **연애 경험 부족**: 첫 연애를 시작하려는데 어떻게 해야 할지 모름
- **표현 방법 모름**: 좋아하는 사람이 있어도 어떻게 다가가야 할지 막막함
- **관계 불안**: 상대방의 마음을 몰라서 불안하고 혼란스러움
- **실패 두려움**: 거절당할까봐 무서워서 시도조차 못함
- **자기 이해 부족**: 나는 어떤 사람인지, 어떤 연애를 원하는지 모름

## 1차 MVP 구현 계획

이번 MVP에서는 **핵심기능 1: 나 사용 설명서 (Level 1)**만 구현하며, 자기 자신을 알아가는 것으로 연애의 첫 시작을 준비합니다.

### 핵심 기능: 나 사용 설명서 (Level 1)

**목표**: "나는 어떤 사람일까?"라는 질문에 답하며 자기 이해를 돕기

**구성**:

1. **감정 패턴 (Section 1)**

   - 나의 평소 상태는? (언제 가장 편안한지)
   - 스트레스 받을 때 나타나는 신호는?
   - 회복하려면 무엇이 필요한지?

2. **갈등 스타일 (Section 2)**

   - 갈등 상황에서 나는 어떻게 반응하나?
   - 어떤 사과를 진심으로 받아들이나?

3. **핵심 가치관 (Section 3)**
   - 절대 타협할 수 없는 가치는? (Non-Negotiables)
   - 나를 행복하게 만드는 작은 것들은? (Joy Triggers)

### 완성 후 제공되는 것

**Love ID 카드 발급**:

- 나 사용 설명서를 기반으로 생성되는 시각적 카드
- 나의 연애 스타일, 주요 특징, 감정적 준비도 표시
- 친구들과 공유하거나 소개팅 전 자기소개 용도로 활용

**궁합 분석** (간단 버전):

- 상대방의 Love ID와 비교하여 궁합 점수 제공
- 갈등 스타일 조합 분석 (예: 불안형 + 회피형 = 주의 필요)
- 공유하는 가치관 하이라이트

**대화 주제 추천**:

- 두 사람의 사용 설명서를 기반으로 함께 이야기하면 좋을 주제 제안
- 궁합이 낮아도 서로를 이해할 수 있도록 가이드 제공

### MVP의 핵심 가치

1. **자기 이해**: "나는 이런 사람이구나" 깨닫기
2. **불안 완화**: 객관적 데이터로 "이 정도면 괜찮아" 안심시키기
3. **행동 가이드**: "다음엔 이렇게 해보세요" 구체적 조언
4. **자신감 부여**: "나도 할 수 있어" 느끼게 하기

### 톤앤매너

- ❌ "데이터를 분석하세요", "관계를 건축하세요"
- ✅ "천천히 알아가요", "괜찮아요, 함께 해봐요"
- 따뜻하고 격려하는 친구 같은 느낌
- 실패를 두려워하지 않도록 응원하는 메시지
