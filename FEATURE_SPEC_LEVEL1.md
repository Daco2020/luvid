# 핵심 기능 1: 나 사용 설명서 (Level 1) - 상세 기획서

> "연애가 서툴러도 괜찮아요. 천천히, 나를 알아가는 것부터 시작해요."

## 1. 기획 의도

### 1.1 왜 이 기능이 필요한가?

우리의 타깃 유저들(민지, 준호, 수빈, 지훈, 예린)은 공통적으로 **"나는 어떤 사람일까?"**라는 질문에 답하지 못합니다.
기존 심리 테스트들은 **MBTI, 애착유형** 같은 라벨만 붙여줄 뿐, **"그래서 나는 연애에서 어떻게 행동해야 하는가?"**에 대한 답은 주지 못합니다.

Luvid는 관계 심리학(Window of Tolerance, Polyvagal Theory)에 기반하여 사용자 자신의 **'작동 방식(Operating Instructions)'**을 체계화합니다.

### 1.2 Level 1의 목표

**완료 후 사용자가 얻는 것**:

1. **자기 이해**: "나는 스트레스 받으면 동굴로 들어가는구나" (신경계 반응 이해)
2. **소통 도구**: "그럴 땐 나를 그냥 둬줘"라고 말할 수 있는 근거 확보
3. **관계 효능감**: 내 감정을 내가 조절할 수 있다는 자신감

**정체성 호칭 부여**:

- 완료 시: **"준비된 연인(Ready-to-Date)"** 뱃지 획득

## 3. 섹션 1: 감정 패턴 발견하기 (Maintenance Manual)

### 3.1 목표

사용자의 **'윈도우 오브 톨러런스(Window of Tolerance)'**를 파악하여, 최적의 상태를 유지하고 스트레스 상황에서 회복하는 **'유지 보수 매뉴얼'**을 완성하는 것.

### 3.2 심리학적 배경

- **윈도우 오브 톨러런스 (Dan Siegel)**: 각성의 최적 영역. 이 범위를 벗어나면 과각성(분노/불안) 혹은 저각성(우울/무기력) 상태가 됨.
- **폴리베이갈 이론 (Stephen Porges)**: 신경계의 3단계 반응 (사회적 참여 -> 가동화 -> 부동화).

### 3.3 질문 구성 (총 10문항)

**[Part 1: 기준점 및 에너지 (Operating Environment)]**

#### Q1. 나의 연료 (Energy Source)

```
질문: 당신이 가장 '나다운' 상태로 충전되는 순간은 언제인가요?

A) 혼자 조용한 공간에서 쉴 때 (Solitude)
   → 내향형 충전
B) 친한 친구 1-2명과 깊은 대화를 할 때 (Intimacy)
   → 관계형 충전
C) 밖에서 사람들과 어울리고 활동할 때 (Social)
   → 외향형 충전
D) 좋아하는 취미나 몰입할 거리가 있을 때 (Activity)
   → 활동형 충전
```

#### Q2. 최적의 시간 (Optimal Time)

```
질문: 중요한 대화를 하기에 가장 좋은, 당신의 '골든 타임'은?

A) 머리가 맑은 '오전 시간'
B) 일과를 마친 '저녁 시간'
C) 배부르고 여유로운 '식사 후'
D) 주말이나 휴일 같은 '쉬는 날'
```

---

**[Part 2: 경고 신호 및 트리거 (Triggers & Warning Signs)]**

#### Q3. 외부 트리거 (External Triggers)

```
질문: 데이트 중, 당신의 기분을 순식간에 망치는 것은? (가장 예민한 부분)

A) 시끄러운 소음이나 복잡한 인파 (감각 과부하)
B) 배고픔이나 더위/추위 (신체적 불편함)
C) 계획이 틀어지거나 갑자기 변동될 때 (통제력 상실)
D) 상대방이 핸드폰만 볼 때 (무시당하는 느낌)
```

#### Q4. 나의 경고등 (Warning Signs)

```
질문: 스트레스를 받기 시작하면, 겉으로 드러나는 당신의 첫 반응은?

A) 말이 없어지고 표정이 굳는다 (Shutdown)
   → 저각성 초기 신호
B) 목소리가 커지거나 말이 빨라진다 (Mobilization)
   → 과각성 초기 신호
C) 한숨을 쉬거나 안절부절못한다 (Anxiety)
   → 불안 반응
D) 냉소적이거나 비꼬는 말투가 나온다 (Defense)
   → 방어 기제
```

---

**[Part 3: 반응 패턴 (Fight, Flight, Freeze)]**

#### Q5. 불확실성 반응 (Uncertainty)

```
상황: 어제 데이트 후 "즐거웠어"라고 보냈는데, 하루가 지나도 답장이 없습니다. (읽음 표시 없음)

A) "바쁘겠지" 하고 내 할 일 한다. (Secure)
   → 안정적 처리
B) "내가 뭐 실수했나?" 계속 대화창을 확인한다. (Anxious)
   → 불안 반응 (과각성)
C) "답장 안 하면 나도 안 해" 하고 번호를 지운다. (Avoidant)
   → 회피/차단 반응
D) "왜 답장이 없어?"라고 문자나 전화를 바로 한다. (Protest)
   → 저항 반응
```

#### Q6. 과각성 반응 (Hyper-arousal)

```
상황: 연인과 말다툼 중, 감정이 주체할 수 없이 격해졌습니다. 이때 당신의 충동은?

A) 상대방을 이기기 위해 논리적으로 따진다. (Fight)
B) 화를 참지 못하고 소리를 지르거나 물건을 던지고 싶다. (Explosion)
C) "그만해!"라고 외치고 자리를 박차고 나간다. (Flight)
D) 눈물이 터져서 말을 잇지 못한다. (Flood)
```

#### Q7. 저각성 반응 (Hypo-arousal)

```
상황: 너무 지치고 힘들어서 에너지가 바닥났을 때(방전), 당신의 모습은?

A) 아무것도 안 하고 하루 종일 누워만 있는다. (Freeze)
B) 상대방의 말(위로)조차 듣기 싫고 귀찮다. (Withdrawal)
C) 멍하니 있거나 현실감이 없어진다. (Dissociation)
D) 무조건적인 지지와 응원만 받고 싶다. (Regression)
```

---

**[Part 4: 회복 매뉴얼 (Regulation & Recovery)]**

#### Q8. 자기 조절 (Self-Regulation)

```
질문: 격해진 감정을 스스로 진정시키는 가장 효과적인 방법은?

A) 혼자만의 시간을 가지며 동굴로 들어간다. (Space)
B) 심호흡을 하거나 찬물로 세수한다. (Physiological)
C) 맛있는 걸 먹거나 좋아하는 영상을 본다. (Sensory)
D) 감정을 글로 쓰거나 친구에게 털어놓는다. (Expression)
```

#### Q9. 상호 조절 (Co-Regulation)

```
질문: 당신이 힘들 때, 연인이 해주길 바라는 '최고의 위로'는?

A) "무슨 일이야?" 하고 끝까지 들어주기 (Listening)
B) 아무 말 없이 꽉 안아주기 (Physical Touch)
C) "내가 해결해줄게" 하고 문제 처리해주기 (Act of Service)
D) "잠시 혼자 쉴래?" 하고 배려해주기 (Respecting Space)
```

#### Q10. 접근 금지 (Do Not Disturb)

```
질문: 기분이 안 좋을 때, 이것만은 절대 하지 말아야 할 것은?

A) "왜 저기아?" 하고 계속 이유 캐묻기 (Intrusion)
B) "별거 아니네" 하고 내 감정 축소하기 (Dismissal)
C) "웃어봐" 하고 분위기 띄우려 하기 (Invalidation)
D) 논리적으로 따지거나 조언하기 (Lecturing)
```

## 5. 섹션 3: 핵심 가치관 (Values)

### 5.1 목표

관계의 나침반이 될 **Top 3 가치**와 **딜브레이커(Dealbreaker)** 식별.

### 5.2 심리학적 배경

- **가치 카드 분류 (Values Card Sort)**: 핵심 가치 도출 방법론.
- **딜브레이커 (Dealbreakers)**: 관계 종료를 결정짓는 치명적 요인.

### 5.3 질문 구성 (총 10문항)

**[Part 1: 인생의 우선순위]**

#### Q1. 가치관 월드컵 (Top 3 Selection)

```
질문: 다음 16가지 가치 중, 당신의 삶에서 가장 중요한 3가지는? (선택형)

[가치 목록 예시]
성장, 안정, 자유, 신뢰, 유머, 사랑, 건강,
부(Wealth), 공헌, 인정, 모험, 정직, 가족,
종교, 소통, 책임
```

#### Q2. 딜레마 시뮬레이션: 커리어 vs 사랑

```
상황: 연인이 해외 발령을 받았습니다. 당신은 한국에서 중요한 커리어 기회가 있습니다.
선택은?

A) 연인을 따라간다 (사랑 우선)
B) 장거리 연애를 하더라도 내 커리어를 지킨다 (자아실현 우선)
C) 절충안을 찾을 수 없다면 헤어진다 (현실적 판단)
D) 연인에게 가지 말라고 설득한다 (관계 중심)
```

#### Q3. 딜레마 시뮬레이션: 가족 vs 개인

```
상황: 명절 연휴, 연인은 "우리 집에 가자"고 하고 당신은 "푹 쉬고 싶다"고 합니다.

A) "그래도 명절인데 가야지" (의무/관계 중시)
B) "이번엔 각자 쉬자" (개인/자율성 중시)
C) "너만 다녀와, 나는 쉴게" (독립성 중시)
D) "짧게 들렀다 오자" (타협 중시)
```

**[Part 2: 라이프스타일 및 딜브레이커]**

#### Q4. 재정 관리 선호 (Money Management)

```
질문: 결혼이나 장기 동거 시, 선호하는 돈 관리 방식은?

A) 모든 수입을 합쳐서 공동 관리 (완전 통합)
B) 생활비 통장만 만들고, 나머지는 각자 관리 (반반)
C) 수입은 각자 관리하고 지출만 분담 (완전 독립)
D) 더 많이 버는 사람이 더 많이 부담 (능력 비례)
```

#### Q5. 주말 여가 방식 (Leisure)

```
질문: 이상적인 주말 데이트 빈도는?

A) 토/일 48시간 내내 붙어 있기 (융합)
B) 토요일은 데이트, 일요일은 각자 휴식 (균형)
C) 주 1회 정도 짧고 굵게 만나기 (독립)
D) 그때그때 기분 따라 다름 (유연)
```

#### Q6. 연락 빈도 (Communication Frequency)

```
질문: 당신이 생각하는 '적절한 연락'의 기준은?

A) 눈 뜨고 감을 때까지 모든 일상 공유 (실시간)
B) 틈날 때마다 카톡, 자기 전 통화 (수시)
C) 급한 용건 위주, 하루 1-2번 정리해서 (필요시)
D) 연락보다는 직접 만났을 때 집중 (오프라인 중심)
```

#### Q7. 이성 친구 허용 범위 (Boundaries)

```
질문: 연인의 '여사친/남사친', 어디까지 허용?

A) 단둘이 식사/술 절대 불가 (보수적)
B) 낮에 커피나 밥은 OK, 술은 NO (중도)
C) 단둘이 만나도 연락만 되면 OK (개방적)
D) 서로 터치하지 않음, 믿음 문제 (자유방임)
```

**[Part 3: 절대 불가 항목 (Dealbreakers)]**

#### Q8. 치명적인 결함 (Behavioral)

```

A) 폭언/욕설 (언어 폭력)
B) 잠수/연락 두절 (회피)

```

#### Q9. 가치관 충돌 (Ideological)

```

A) 종교 강요 (신념 충돌)

```

#### Q10. 관계의 비전 (Vision)

```

A) 서로의 보호자가 되어주는 가족 같은 관계

C) 친구처럼 편안하고 즐거운 베프 관계
D) 언제나 설레고 열정적인 연인 관계
```
