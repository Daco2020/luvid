# 핵심 기능 1: 나 사용 설명서 (Level 1) - 상세 기획서

> "연애가 서툴러도 괜찮아요. 천천히, 나를 알아가는 것부터 시작해요."

## 1. 기획 의도

### 1.1 왜 이 기능이 필요한가?

우리의 타깃 유저들(민지, 준호, 수빈, 지훈, 예린)은 공통적으로 **"나는 어떤 사람일까?"**라는 질문에 답하지 못합니다.

- 민지는 "내가 어떤 연애 스타일인지" 모릅니다
- 준호는 "왜 항상 2-3번 만나고 끝나는지" 모릅니다
- 수빈은 "내가 진짜 원하는 게 뭔지" 모릅니다
- 지훈은 "왜 같은 실수를 반복하는지" 모릅니다
- 예린은 "나한테 맞는 사람의 기준"이 없습니다

기존 심리 테스트들은 **MBTI, 애착유형** 같은 라벨만 붙여줄 뿐, **"그래서 나는 연애에서 어떻게 행동해야 하는가?"**에 대한 답은 주지 못합니다.

**Luvid의 차별점**:

- ❌ "당신은 불안형 애착입니다" (라벨링)
- ✅ "당신은 연락이 안 오면 불안해지는 사람이에요. 그럴 땐 이렇게 해보세요" (실용적 인사이트)

### 1.2 Level 1의 목표

**완료 후 사용자가 얻는 것**:

1. **자기 이해**: "아, 나는 이런 사람이구나" (3가지 핵심 패턴 발견)
2. **안심**: "이런 게 나만 그런 게 아니구나" (정상화)
3. **기대감**: "이걸 알면 다음 연애는 다를 수 있겠다" (희망)

**정체성 호칭 부여**:

- 완료 시: **"연애할 준비가 된 사람"** 뱃지 획득
- 화면 표시: "당신은 이제 자신의 마음을 아는 사람이에요 💭"

## 2. 전체 구조 (3 Sections, ~20분)

### 섹션 구성 철학

| 섹션       | 주제        | 입력 방식              | 소요 시간 | 난이도      | 심리학적 근거                           |
| ---------- | ----------- | ---------------------- | --------- | ----------- | --------------------------------------- |
| **섹션 1** | 감정 패턴   | 시나리오 선택          | 5-7분     | ⭐ 쉬움     | 구체적 상황이 답하기 쉬움               |
| **섹션 2** | 갈등 스타일 | 슬라이더 + 이미지      | 6-8분     | ⭐⭐ 보통   | 시각적 표현으로 추상 개념 구체화        |
| **섹션 3** | 핵심 가치관 | 우선순위 정렬 + 주관식 | 7-9분     | ⭐⭐⭐ 깊음 | 가치 충돌 시뮬레이션으로 진짜 가치 도출 |

**진행 방식**:

- 각 섹션 완료 시 로컬스토리지 자동 저장
- 중간에 나가도 이어서 가능
- 섹션 간 전환 시 "잠깐 쉬어가기" 화면 - 방금 완료한 섹션이 어땠는지 묻고 간단한 설명(비유, 은유 활용)을 제공, 그리고 다음 섹션에 대한 후킹한 한 문장 보여주기

## 3. 섹션 1: 감정 패턴 발견하기 (5-7분)

### 3.1 목표

사용자의 **스트레스 반응 패턴**과 **회복 방식**을 파악

### 3.2 심리학적 배경

- **윈도우 오브 톨러런스** (Dan Siegel): 사람마다 감정 조절 범위가 다름
- **폴리베이갈 이론** (Stephen Porges): 스트레스 시 과각성/저각성 반응

### 3.3 차별화 포인트

기존 테스트: "당신은 스트레스를 받으면 어떻게 하나요?" (추상적)
**Luvid**: 실제 연애 상황 시나리오 제시 (구체적)

### 3.4 질문 구성 (총 8문항)

#### Q1. 첫 데이트 30분 전

```
상황: 오늘 처음 만나는 사람과 카페 데이트 약속이 있어요.
30분 전, 상대방이 문자를 보냅니다.
"갑자기 일이 생겨서 1시간 늦을 것 같아요. 괜찮을까요?"

당신의 첫 반응은?

A) "괜찮아요! 천천히 오세요 😊"
   → 즉시 답장하고 근처 서점 구경하러 감

B) "아... 그래요? 몇 시쯤 가능할까요?"
   → 답장하고 불안해하며 카톡 확인 반복

C) "네, 알겠습니다"
   → 답장하고 핸드폰 끄고 산책 나감

D) 답장 안 하고 일단 시간 둠
   → "뭐라고 답해야 하지?" 고민하다가 10분 후 답장
```

**측정 목표**:

- A: 수용형 (타인 배려 우선)
- B: 불안형 (확인 욕구)
- C: 독립형 (감정 차단)
- D: 회피형 (결정 회피)

#### Q2. 데이트 중 갑자기 말이 없어진 상대방

```
상황: 데이트 중 상대방이 갑자기 말이 없어지고 표정이 굳었어요.
"무슨 일 있어?"라고 물었더니 "아니, 아무것도 아니야"라고 합니다.

당신은?

A) "그래? 그럼 다행이다!"
   → 그냥 넘어가고 다른 이야기 시작

B) "진짜 괜찮아? 내가 뭐 잘못했어?"
   → 계속 물어보며 불안해함

C) "말하기 싫으면 나중에 말해줘"
   → 더 이상 묻지 않고 조용히 기다림

D) "아... 그렇구나"
   → 어색해서 핸드폰 보기 시작
```

**측정 목표**: 타인의 감정 변화에 대한 반응 패턴

#### Q3. 카톡 답장이 6시간째 없음

```
상황: 어제 데이트 후 "오늘 재밌었어!"라고 문자 보냈는데
6시간째 답장이 없습니다. (읽음 표시 없음)

당신의 마음 상태는?

A) "바쁘겠지 뭐"
   → 별로 신경 안 쓰고 내 할 일 함

B) "뭐 잘못했나? 재미없었나?"
   → 계속 핸드폰 확인하며 불안함

C) "답장 안 하는 사람이면 관심 없는 거지"
   → 실망하고 기대 접음

D) "나도 답장 천천히 해야지"
   → 전략적으로 대응 계획 세움
```

**측정 목표**: 불확실성에 대한 내성

#### Q4. 싸운 후 다음 날 아침

```
상황: 어젯밤 사소한 일로 다퉜어요.
서로 "잘 자"도 없이 그냥 잤습니다.
다음 날 아침, 당신은?

A) 먼저 "어제는 미안해"라고 문자 보냄
   → 빨리 풀고 싶음

B) 상대방이 먼저 연락할 때까지 기다림
   → "내가 왜 먼저 연락해?"

C) "어제 밥 먹었어?"라고 가볍게 문자
   → 직접적인 사과는 어색함

D) 아무 연락도 안 함
   → "시간이 지나면 자연스럽게 풀리겠지"
```

**측정 목표**: 갈등 후 화해 패턴

#### Q5. 스트레스 받을 때 나는?

```
상황: 오늘 하루가 최악이었어요.
상사한테 혼나고, 프로젝트는 꼬이고, 지하철도 놓쳤습니다.
집에 도착한 당신, 연인이 "오늘 어땠어?"라고 물어봅니다.

당신은?

A) 하루 종일 있었던 일을 쏟아내며 하소연
   → "들어줘서 고마워" 기분이 풀림

B) "그냥 좀 힘들었어..." 짧게 말하고 침묵
   → 말하기 싫고 혼자 있고 싶음

C) "괜찮아, 별일 아니야"
   → 걱정 끼치기 싫어서 숨김

D) "오늘 진짜 최악이었어. 근데 지금은 말하기 싫어"
   → 솔직하게 경계 표현
```

**측정 목표**: 취약성 공유 vs 감정 차단 성향

#### Q6. 혼자만의 시간

```
질문: 당신에게 갑자기 주말 이틀이 완전히 비었습니다.
연인은 친구 결혼식 때문에 지방에 갔어요.

당신은 이 시간을?

A) "외롭다..."
   → 친구 붙잡고 약속 잡음

B) "드디어 혼자다!"
   → 집에서 넷플릭스 정주행

C) "뭐 하지?"
   → 계획 없이 어영부영 시간 보냄

D) "이 기회에 밀린 일 해야지"
   → 생산적으로 시간 활용
```

**측정 목표**: 혼자 있는 시간에 대한 태도 (재충전 vs 외로움)

#### Q7. 기분이 안 좋을 때 위로 방식

```
상황: 당신이 기분이 안 좋을 때,
연인이 어떻게 해주면 가장 좋을까요?

A) 말없이 안아주기
   → 스킨십으로 위로받고 싶음

B) "무슨 일이야? 이야기해봐"
   → 경청하고 공감해주기

C) "맛있는 거 먹으러 가자!"
   → 기분 전환시켜주기

D) 그냥 혼자 내버려두기
   → 간섭하지 않는 게 최고
```

**측정 목표**: 선호하는 위로 언어

#### Q8. 에너지 충전 방식

```
질문: 일주일 내내 사람들 만나고 회의하고 정신없었어요.
주말에 당신은?

A) 집에서 혼자 조용히 쉼
   → 아무도 안 만나고 배터리 충전

B) 친한 친구 1-2명이랑 조용한 카페
   → 편한 사람과 수다

C) 동호회/모임 나가서 새로운 사람 만남
   → 사람 만나야 에너지 충전됨

D) 운동/등산/활동적인 것
   → 몸 움직여야 스트레스 풀림
```

**측정 목표**: 내향성/외향성 + 회복 방식

## 4. 섹션 2: 갈등 스타일 이해하기 (6-8분)

### 4.1 목표

사용자의 **갈등 대처 방식**과 **사과 수용 언어** 파악

### 4.2 심리학적 배경

- **TKI 갈등 모드** (Thomas-Kilmann): 5가지 갈등 스타일
- **사과 언어** (Gary Chapman): 5가지 사과 수용 방식
- **고트맨의 4가지 파멸의 기사**: 관계를 망치는 소통 패턴

### 4.3 차별화 포인트

기존 테스트: 객관식 질문만
**Luvid**: 슬라이더 + 이미지 선택으로 시각화

### 4.4 질문 구성 (총 10문항)

#### Q1. 갈등 스타일 슬라이더

```
질문: 연인과 의견이 다를 때, 당신은?

[슬라이더]
━━━━━●━━━━━
피한다          맞선다

왼쪽 끝: "갈등 자체가 싫어요. 웬만하면 피하고 싶어요"
중간: "상황에 따라 다르지만, 중요한 건 이야기해요"
오른쪽 끝: "문제는 바로 풀어야 해요. 피하면 더 커져요"
```

**측정**: 회피형 ↔ 직면형

---

#### Q2. 싸움 중 나의 모습 (이미지 선택)

```
질문: 싸우는 중에 나는 주로?

[이미지 4개 - 일러스트로 표현]

🗣️ "말이 많아져요"
   → 설명하고 설득하려 함

🤐 "말이 없어져요"
   → 담쌓기, 침묵

😤 "목소리가 커져요"
   → 감정적으로 격해짐

🚪 "자리를 피해요"
   → 물리적으로 거리 둠
```

**측정**: 고트맨의 4가지 파멸의 기사 중 어디에 해당하는지

---

#### Q3. 사과 수용 언어 (우선순위)

```
질문: 연인이 잘못했을 때, 어떤 사과가 가장 진심으로 느껴지나요?
(순서대로 드래그해서 정렬해주세요)

[드래그 가능한 카드 5개]

💬 "진심으로 미안해. 내가 잘못했어"
   → 감정적 표현

📝 "내가 ~한 게 잘못이었어. 변명 안 할게"
   → 책임 인정

🎁 "미안해. 내가 어떻게 만회할까?"
   → 보상 제안

🔄 "다시는 이런 일 없도록 ~하겠다고 약속할게"
   → 행동 변화 약속

🙏 "나를 용서해줄 수 있을까?"
   → 용서 구하기
```

**측정**: 5가지 사과 언어 우선순위

#### Q4. 금기 사항 체크리스트

```
질문: 싸울 때 상대방이 이것만은 절대 하지 말았으면 하는 것은?
(해당하는 것 모두 선택)

☐ 과거 일 들춰내기 ("너 지난번에도 그랬잖아")
☐ 인격 공격 ("너는 항상 이기적이야")
☐ 비교하기 ("내 친구 남자친구는 안 그러던데")
☐ 무시하기 (한숨, 눈 흘기기, 비웃기)
☐ 헤어지자는 말 ("그럼 헤어지자")
☐ 잠수 타기 (연락 두절)
☐ 제3자한테 말하기 (친구/가족한테 험담)
```

**측정**: 개인별 트리거 (딜브레이커)

### 7.3 중간 저장 & 이어하기

- 각 섹션 완료 시 자동 저장
- 브라우저 닫아도 다시 돌아오면 이어서 가능
- "나중에 하기" 버튼 → 저장하고 나가기

## 8. UX/UI 원칙

### 8.1 미니멀리즘

- 한 화면에 하나의 질문만
- 불필요한 장식 제거
- 여백을 충분히

### 8.2 따뜻한 톤

- "당신은 틀렸어요" ❌
- "이런 당신도 괜찮아요" ✅

### 8.3 즉각적인 피드백

- 답변 선택 시 부드러운 애니메이션
- "좋아요, 다음 질문이에요" 같은 격려 메시지

### 8.4 진행 가능성 보장

- "정답"은 없음
- 모든 선택이 유효함
- 판단하지 않는 태도
